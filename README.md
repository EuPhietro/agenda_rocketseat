# 📒 Agenda de Contatos (CLI)

## Sobre o Projeto
Este projeto nasceu inicialmente como um desafio do curso de Introdução ao Python da **Rocketseat**. Contudo, decidi transformá-lo em um laboratório pessoal para construir um CRUD (Create, Read, Update, Delete) completo via linha de comando. 

O propósito central desta aplicação foi acelerar minha curva de aprendizado na leitura e interpretação de documentações técnicas oficiais. Para isso, estruturei o sistema utilizando ferramentas consagradas do ecossistema Python, unindo simplicidade, robustez e alta qualidade de código.

## Tecnologias Utilizadas
* **Python** — Linguagem base do ecossistema.
* **Peewee ORM** — Abstração leve e eficiente para a persistência e manipulação de dados relacionais.
* **Rich** — Renderização avançada de tabelas, painéis e componentes visuais interativos (TUI) diretamente no terminal.
* **SQLite** — Banco de dados relacional local e autocontido para armazenamento seguro.

## Aprendizados Chave
* **Arquitetura em Camadas:** Separação clara de responsabilidades isolando estritamente as camadas de Modelos, Repositórios e Serviços.
* **Integridade Referencial:** Implementação prática e segura de relacionamentos entre entidades e deleções em cascata (`ON DELETE CASCADE`).
* **Gestão de Estado em CLI:** Controle de ciclo de vida de componentes visuais, tratamento de concorrência de memória no terminal e mitigação de duplicação de dados.
* **Autonomia Técnica:** Navegação aprofundada em documentações oficiais para engenharia de software defensiva e resolução de problemas complexos de infraestrutura.
